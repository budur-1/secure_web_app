# Secure Web Application (Before & After)

## Overview
This project demonstrates a simple web application secured against common vulnerabilities. It includes two versions:
- **BEFORE:** Vulnerable version for testing.
- **AFTER:** Secured version with all protections applied.

Covered topics:
- SQL Injection
- XSS
- Weak Password Storage
- Access Control (RBAC)
- Encryption
- HTTPS & Secure Sessions
